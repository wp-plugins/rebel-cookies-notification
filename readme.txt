Cookies plugin to display notification to users regarding cookie usage.

== Description ==

This simple cookie notification plugin allows your WordPress site to display a simple and discreet notification bar along the top or bottom of your site.

The Rebel Cookie Notification Plugin allows you to be able to change the colour of the bar, text and cookie page link. Using the settings panel for the 
plugin you can also link an existing page to the notification bar allowing your viewers to read the page explaining your cookie usage.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `RAS_Cookies_Notification` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. (Optional) Configure the plugin through the settings panel which can be found on the left menu at the bottom.
